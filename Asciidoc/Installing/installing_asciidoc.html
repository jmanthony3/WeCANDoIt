1. Software Requirements and Installation Links to Executables/Binaries

To write AsciiDoc markup documents and compile to other formats [1], be able to run the Ruby language [2]. Refer to official AsciiDoc installation instructions [1] for local machine setup.

2. Installing Gems

2.1. Required

  • gem install asciidoctor [1]

2.2. Recommended

Syntax Highlighting [3]

  • gem install rouge

  • gem install pygments.rb

  • gem install coderay

Set the appropriate variable (example): :source-highlighter: rouge

PDF Output [3]

  • gem install asciidoctor-pdf

LaTeX Support for Typesetting Mathematics [4]

  • gem install asciidoctor-latex --pre

BibTeX Files and Citations [5]

  • gem install asciidoctor-bibtex

AsciiDoxy [6]

  • pip3 install --update asciidoxy
